# Buko Website

Teaser landing page for Buko.

## Stack

- React + Vite
- React Three Fiber
- Drei
- Three.js

## Development

Install dependencies:

```sh
npm install
```

Start the local dev server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```
